# Utilisation de l'IA pour générer du code
L'intelligence artificielle est un atout formidable pour gagner du temps. Mais Git reste indispensable pour:
1. Garder un historique de ce que l'IA a fait
2. Revenir en arrière si le code généré casse la page
3. Suivre précisément les modifications apportées (avec git diff) avant de les valider.
J'ai utilisé l'IA pour m'aider sur la compréhension de certains petits scripts, et je les ai versionnés grâce à Git pour toujours avoir le contrôle.
